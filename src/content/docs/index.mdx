## Quick Example

Write PlantUML in your markdown:

````markdown
```plantuml
@startuml
Alice -> Bob: Hello!
Bob --> Alice: Hi there!
@enduml
```
````

And it automatically renders as:

```plantuml
@startuml
Alice -> Bob: Hello!
Bob --> Alice: Hi there!
@enduml
```
